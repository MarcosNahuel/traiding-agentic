## Codebase Patterns
- Hybrid stack: Frontend (Vercel Next.js) + Backend (Docker VPS FastAPI)
- Trading loop: 2 concurrent loops (60s main + 2s fast SL/TP)
- Atomic execution: UPDATE WHERE status=approved → executing (compare-and-swap)
- SL/TP: ATR-based con Chandelier Exit + time stop 48h
- Signals: Technical rules (RSI, ADX, PPO, volume) + ML (LightGBM)
- Risk: 8-check pipeline (5 base + 3 quant)
- DB: Supabase PostgreSQL via PostgREST
- Kill switch: TRADING_ENABLED=false por defecto
